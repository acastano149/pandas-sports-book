5.3 3. Conceptos Clave para la Agregaci√≥n

Antes de los ejercicios, asegur√©monos de entender los m√©todos que utilizar√°s:

Noteüí° El patr√≥n split-apply-combine

La agregaci√≥n en Pandas sigue el patr√≥n split-apply-combine:

  1. Split (.groupby()): Divide el DataFrame en grupos basados en una columna
  2. Apply: Aplica una funci√≥n a cada grupo (.mean(), .sum(), .agg())
  3. Combine: Combina los resultados en un nuevo DataFrame
# Estructura b√°sica
df.groupby('columna_grupos')['columna_valores'].funcion()
Noteüí° .agg() - M√∫ltiples agregaciones

.agg() permite aplicar varias funciones a la vez con nombres personalizados:

df.groupby('col').agg(
    Nombre_resultado = ('columna_origen', 'funcion')
)

Funciones disponibles: 'mean', 'sum', 'count', 'min', 'max', 'std', etc.

Noteüí° .transform() - Agregar sin colapsar

A diferencia de .agg() que colapsa el DataFrame, .transform() devuelve un valor para cada fila original. Es como a√±adir una columna con la media del grupo a cada fila.

# A√±ade la media del grupo a cada fila

Noteüí° .rolling() - Ventanas deslizantes

.rolling(window=n) calcula estad√≠sticas sobre una ventana m√≥vil de n observaciones. Muy √∫til para detectar tendencias y suavizar series temporales.

df['Media_Movil_7dias'] = df['Valor'].rolling(window=7).mean()

Las primeras n-1 observaciones ser√°n NaN porque no hay suficientes datos previos.

Noteüí° .shift() - Acceder a valores anteriores/posteriores

.shift(n) desplaza los valores n posiciones. Es el equivalente a lag() en R.

  • .shift(1) ‚Üí valor anterior (lag)
  • .shift(-1) ‚Üí valor siguiente (lead)
df['Dist_Ayer'] = df['Distancia'].shift(1)
df['Cambio'] = df['Distancia'] - df['Dist_Ayer']
